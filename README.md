# 🚀 Personal Portfolio Website

A modern, responsive, and interactive personal portfolio website built with **HTML, CSS, and JavaScript**.  
This website is designed to introduce personal information, education background, technical skills, selected projects, achievements, and contact information in a clean and professional way.

## 📋 Project Description

This project is a static personal portfolio website for **Nguyen Pham Chi Bao**, a Data Science student who is interested in Data Analytics, Data Engineering, Machine Learning, Artificial Intelligence, and Web Development.

The website helps present personal branding, academic information, project experience, skills, certificates, and contact channels. It also includes interactive features such as dark mode, typing animation, particles background, smooth scrolling, project filtering, and scroll animations.

## 📦 Main Files

### 1. `index.html`

The main HTML file that contains the full structure of the portfolio website.

It includes the following sections:

- Navbar
- Hero section
- About section
- Education section
- Projects section
- Awards & Achievements section
- Skills section
- Contact section
- Footer

This file also connects external libraries such as Font Awesome, Google Fonts, Typed.js, Particles.js, jQuery, and local files including `styles.css` and `script.js`.

### 2. `styles.css`

The main CSS file used to style the whole website.

It controls:

- Layout and spacing
- Typography
- Color system
- Light mode and dark mode
- Navbar design
- Hero section layout
- Project cards
- Skills cards
- Awards cards
- Contact form
- Responsive design
- Hover effects and animations

The design uses CSS variables, making it easier to manage colors and theme changes.

### 3. `script.js`

The JavaScript file that adds interactivity to the website.

Main functions include:

- Dark mode and light mode toggle
- Saving selected theme in local storage
- Typed text animation in the hero section
- Particles background animation
- Smooth scrolling between sections
- Mobile navigation menu toggle
- Project category filtering
- Fade-in animation when scrolling
- Hero background parallax effect

### 4. `images/`

This folder contains all images used in the portfolio website.

It includes the profile image and preview images for selected projects.

### 5. `.github/workflows/static.yml`

This file is used for GitHub Actions deployment.

It helps deploy the portfolio automatically as a static website through GitHub Pages.

## ✨ Key Features

- Modern and clean portfolio design
- Fully responsive layout
- Fixed navigation bar
- Smooth scrolling effect
- Dark mode and light mode support
- Typing animation in the hero section
- Animated particles background
- Project filtering by category
- Fade-in animation on scroll
- Contact form integration with Formspree
- Social media links
- Mobile-friendly navigation menu
- GitHub Pages deployment support

## 📝 Notes

- This is a static website, so it does not require a backend server.
- The contact form uses Formspree to receive messages.
- The selected theme is saved in local storage.
- The project can be deployed easily using GitHub Pages.
- All main interactions are handled in `script.js`.
- All visual styling is managed in `styles.css`.
